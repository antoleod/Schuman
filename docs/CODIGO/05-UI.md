# UI

## `src/Schuman.Automation/Main.ps1`
- Launcher principal.
- Botones: `Dashboard`, `Generate`.
- Usa procesos hijo con `-NoLogo -NoProfile -NonInteractive` para evitar banner.
- Incluye loading dialogs con animacion (timers).

## `src/Schuman.Automation/UI/DashboardUI.ps1`
- Dashboard operativo:
  - filtro live por texto,
  - click fila abre ServiceNow,
  - doble click selecciona tarea y accion (check-in/check-out),
  - ajustes de notas por defecto,
  - update de estado en SNOW + Excel.

## `src/Schuman.Automation/UI/GenerateUI.ps1`
- UI clean de generacion.
- Footer de 4 botones:
  - `Generate Documents`
  - `Dashboard (Check-in / Check-out)`
  - `Open Output Folder`
  - `Show Log`
- `Generate Documents` ejecuta callback real desde `Main.ps1`.

## `src/Schuman.Automation/UI/Theme.ps1`
- Helpers visuales reutilizables (fuente, tarjetas, colores).
