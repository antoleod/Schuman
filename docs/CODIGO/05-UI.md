# UI

## `src/Schuman.Automation/Main.ps1`
- Main launcher.
- Buttons: `Dashboard`, `Generate`.
- Uses child processes with `-NoLogo -NoProfile -NonInteractive` to avoid banner noise.
- Includes loading dialogs with timer-based animation.

## `src/Schuman.Automation/UI/DashboardUI.ps1`
- Operational dashboard:
  - live text filter,
  - row click opens ServiceNow,
  - double-click selects task and action (check-in/check-out),
  - updates state in ServiceNow + Excel.

## `src/Schuman.Automation/UI/GenerateUI.ps1`
- Clean generation UI.
- Footer buttons:
  - `Generate Documents`
  - `Dashboard (Check-in / Check-out)`
  - `Open Output Folder`
  - `Show Log`
- `Generate Documents` runs the real callback from `Main.ps1`.

## `src/Schuman.Automation/UI/Theme.ps1`
- Reusable visual helpers (font, cards, button shape, emergency-close helper).
